## 🔍 Overview

MediaExport Master is a 💪 powerful and 🤩 intuitive WordPress plugin that simplifies the process of exporting 🖼️ media library data. With advanced 🎛️ filtering options and seamless 🌀 Tailwind CSS integration, the plugin is designed for 👩‍💻 developers and 🧑‍💻 non-technical users alike.

- **📝 Custom Metadata Export**: Choose from various metadata fields such as 🏷️ Title, 🖼️ Alt Text, 🖊️ Description, and more.

## 📥 Installation Guide

Follow these steps to install MediaExport Master:

1. **⬇️ Download the Plugin**:
   - Obtain the plugin 📦 ZIP file from the official WordPress 🏠 repository or GitHub.

2. **📤 Upload the Plugin**:
   - Log in to your WordPress 🖥️ dashboard.
   - Navigate to `🔌 Plugins > ➕ Add New`.
   - Click `📤 Upload Plugin` and select the 📦 ZIP file.

3. **✅ Activate the Plugin**:
   - After 🛠️ installation, click `✅ Activate` to enable MediaExport Master.

4. **✔️ Verify Installation**:
   - Navigate to `🖼️ Media > MediaExport Master` in the WordPress 🏠 admin menu.

---

## 🛠️ Usage Instructions

1. **🔑 Access the Plugin**:
   - Go to the WordPress dashboard and navigate to `🖼️ Media > MediaExport Master`.

2. **✔️ Select Metadata Fields**:
   - Choose the metadata fields you want to include in your export (e.g., 🆔 ID, 🏷️ Title, 🌐 URL).

3. **🔍 Filter by Author**:
   - Use the dropdown to filter 🖼️ media by 👤 author or select `🧑‍💻 All` for all 🖼️ media items.

4. **📂 Choose Export Type**:
   - Select between `📄 CSV File` or `📜 Output here`.

5. **📤 Export Data**:
   - Click `🚀 Export Now` to generate the 📄 file or display results.

# MediaExport Master

MediaExport Master is a powerful WordPress plugin that allows you to export your media library data with customizable metadata fields and filters. The plugin provides options to filter media items by author, select specific metadata fields, and export data as a CSV file or display it directly on-screen.


## Features

- **Customizable Metadata Fields**: Select metadata fields such as ID, Title, File Name, Caption, Alt Text, Description, URL, Date Uploaded, and Type.
- **Filter by Author**: Export media data specific to an author or include all authors.
- **Export Formats**: Choose between exporting as a CSV file or displaying results on-screen.
- **Tailwind CSS Integration**: Styled with Tailwind CSS for a modern and responsive UI.
- **Dynamic Author Sidebar**: Includes a support and developer info section in the admin page sidebar.

## Installation

1. Download the plugin ZIP file.
2. Log in to your WordPress admin dashboard.
3. Navigate to `Plugins > Add New`.
4. Click `Upload Plugin` and select the downloaded ZIP file.
5. Click `Install Now` and activate the plugin.

## Usage

1. Go to the plugin admin page by navigating to `Media > MediaExport Master`.
2. Select the metadata fields you want to include in the export.
3. Filter by author if necessary.
4. Choose the export type: `CSV File` or `Output here`.
5. Click the `Export Now` button to generate the results.
